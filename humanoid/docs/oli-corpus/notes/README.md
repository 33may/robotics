# Oli Corpus Notes

This folder is for hand-written analysis and interpretation only.

Files under `sources/` are regenerated upstream content and must not be edited by hand. Any derived notes must cite their source sections with `oli-corpus://...` URIs and remain separate from upstream source truth.

MCP search excludes notes by default. Agents must pass `include_notes=true` when they intentionally want curated notes mixed with source results.
